# CookedAI-Privacy

Privacy policy page for the **CookedAI** mobile app, hosted via GitHub Pages.

## Live URL

Once GitHub Pages is enabled the policy will be available at:  
`https://baribarigood.github.io/CookedAI-Privacy/`

## How to enable GitHub Pages

1. Go to **Settings → Pages** in this repository.
2. Under **Source**, select **Deploy from a branch**.
3. Choose the `main` branch and `/ (root)` folder, then click **Save**.
4. GitHub will build and publish the site within a minute or two.

## How to add the Termly privacy policy

1. Open `index.html` in this repository.
2. Find the comment block that reads:
   ```
   <!-- PASTE YOUR TERMLY PRIVACY POLICY HTML HERE -->
   ```
3. Delete that comment and paste the HTML snippet copied from your Termly dashboard in its place.
4. Commit and push — GitHub Pages will update automatically.
